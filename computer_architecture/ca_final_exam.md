# 计算机体系结构 - 考试真题整理 (2026.1.5)

## 第一部分：填空与简答题 (乱序版)

1.  **冯·诺依曼结构的本质是？**
    *   答案：存储程序 + 指令驱动

2.  **体系结构四大设计原则：**
    *   平衡性 (Balance)
    *   局部性 (Locality)
    *   并行性 (Parallelism)
    *   虚拟化 (Virtualization)

3.  **指令系统四大设计原则：**
    *   兼容性 (Compatibility)
    *   通用性 (Generality)
    *   高效性 (Efficiency)
    *   安全性 (Security)

4.  **三个并行性是什么？**
    *   指令级并行 ILP、数据级并行 DLP、线程级/任务级并行 TLP

5.  **RISC 的 \_\_\_\_\_\_ 减少了指令相关性？**
    *   存算分离

6.  **0x4Df70000 ? 转十进制**
    *   *(注：题目要求解析该十六进制数，通常考察 IEEE 754 浮点数表示)*

7.  **DDR4-3200 的相关参数：**
    *   频率是多少？ 
    *   带宽是多少？
    *   DDR 的英文全称是？ *(Double Data Rate)*

8.  **四路组相联 Cache 的 Tag, Offset, Index 位数计算？**
    *   *(需根据具体的 Cache 大小和块大小计算，题目考察计算公式)*

9.  **TLB (Translation Lookaside Buffer) 的作用是什么？**
    *   *(加速虚拟地址到物理地址的转换)*

10. **ESI 模型状态转换：**
    *   **场景**：处理器有四个，共享行 X。
    *   **初始态**：处理器状态分别为 INV (无效), SHD (共享), SHD, SHD。
    *   **动作**：处理器 1 发出 Store (写) 请求。
    *   **问题**：状态变为？

11. **CPU 和 IO 的两种同步方式是？**
    *   *(查询/轮询、中断)*

12. **CPU 和 IO 的两种通信方式是？**
    *   *(PIO、DMA)*

13. **ROB (Reorder Buffer) 的作用是什么？**
    *   *(使指令顺序提交，支持精确异常)*

14. **用于检测未初始值并允许在结果为未定义数字时程序继续执行的数是？**
    *   答案：NaN (Not a Number)

15. **非门 (NOT Gate) 由哪两个晶体管构成？**
    *   *(一个 PMOS 和一个 NMOS)*

16. **线程之间同步的两种方式是？**
    *   *(是什么呢)*

17. **提高处理器性能的几个技术？**
    *   流水线 (Pipeline)
    *   超标量 (Superscalar)
    *   乱序执行 (Out-of-Order)
    *   *(预测、缓存等)*

18. **按指令长度把指令集分为哪三种？**
    *   RISC (精简指令集)
    *   CISC (复杂指令集)
    *   VLIW (超长指令字)

19. **TLB/Cache 替换的三种策略是？**
    *   随机替换 (Random)
    *   \_\_\_\_\_\_ *(LRU - 最近最少使用)*
    *   \_\_\_\_\_\_ *(FIFO - 先进先出)*

20. **处理器三个相关是什么？谁是真相关？**
    *   RAW (写后读 - 真相关)
    *   WAR (读后写 - 反相关)
    *   WAW (写后写 - 输出相关)

21. **写出两个模拟器：**
    *   *(例如：Gem5, QEMU, SimpleScalar)*

22. **写出两个共享内存并行编程模型：**
    *   *(例如：OpenMP, Pthreads)*

23. **中断的两种方式是中断线和什么？**
    *   *(消息中断)*

24. **功耗主要分为什么和什么？**
    *   动态功耗 (Dynamic Power)
    *   静态/漏电流功耗 (Static/Leakage Power)

25. **ERTN 指令的作用除了跳转外还有什么？**
    *   *(注：ERTN 为 LoongArch/MIPS 特有指令，作用是异常返回。除了跳转回 EPC，还需要恢复特权等级和中断使能状态)*

26. **内存提高效率的办法？**
    *   行缓存局部性 (Row Buffer Locality)
    *   Bank 级并行度 (Bank-level Parallelism)

27. **精确异常是什么？**
    *   *(指发生异常时，受害者指令前的所有指令都已完成，受害者及其后的指令都未修改状态，且异常处理后能正确恢复执行)*

## 第二部分：计算与分析题

1.  **MFLOPS 性能计算**
    *   计算机 A: $12 \times 10^9$ Instructions, 耗时 60s
    *   计算机 B: $4 \times 10^9$ Instructions, 耗时 50s
    *   *问题：计算 MFLOPS 或比较性能。*

2.  **CMOS 电路图绘制**
    *   画出逻辑表达式：$(a | b) \& (c | d)$ 的 CMOS 电路。
    *   *(提示：需画出上拉网络 PMOS 和下拉网络 NMOS)*

3.  **转移预测准确率计算**
    *   **代码片段**：
        ```c
        for (i = 0; i < 997; i++) { // 对应转移指令 A
            if (i % 4 == 0) { // 对应转移指令 B
                ...
            }
        }
        ```
    *   **问题**：计算 A 指令和 B 指令的转移预测准确率（分别使用 1位 BHT 和 2位 BHT）。

4.  **多处理器一致性 (顺序执行结果)**
    *   操作：
        *   $P_{1}$: `X = 1; Print Z`
        *   $P_2$: `Y = 1; Print X`
        *   $P_3$: `Z = 1; Print Y`
    *   **问题**：顺序一致性（Sequential Consistency）下可能的结果。

5.  **流水线时空图**
    *   **问题**：画出某段指令序列的流水线时空图。

6.  **Cache/TLB 缺失次数计算**
    *   **代码片段 (矩阵乘法)**：
        ```c
        double A[512][256];
        double B[256][512];
        double C[256][256];
        
        // 三重循环
        for (i)
          for (j)
            for (k)
              C[i][j] = A[i][k] * B[k][j];
        ```
    *   **问题**：计算 TLB Refill, PIL (Page Invalid Load) 和 PIS (Page Invalid Store) 的次数。

7.  **32位 Booth 乘法与电路设计**
    *   **问题**：
        1.  画出全加器 (Full Adder) 的门级电路。
        2.  画出 Booth 乘法器的结构。
        3.  画出超前进位加法器 (CLA) 的门级电路。
        4.  画出华莱士树 (Wallace Tree) 的门级电路。
        5.  根据上述部件计算延迟。
