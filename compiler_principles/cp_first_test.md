# 2026年CP第一次小测
共三大题
每年第一题会发生变动，后面的题应该比较固定

**第一题（30分）**

1. 举例说明DFA在词法分析和语法分析中的应用
2. 给出一个描述“人民币余额”的正则表达式，要求：
	- 余额以`¥`符号开始
	- 每次显示的最小单位为分
	- 自动插入千分位符号
	比如识别`¥0.00`，`¥1.23`，`¥12,000,000.00`，拒绝`¥4.321`和`¥56,99,00`
3. 根据正则表达式构造NFA（按照Thompson算法）

**第二题（35分）**

`V -> O | str | number`

`O -> {} | {S}`

`S -> P | S, P`

`P -> str:V`

1. 消除左递归
2. 提取公因子
3. 计算`FIRST`, `FOLLOW`集合
4. 给出预测分析表
5. 他是LL(1)文法吗？为什么？

**第三题（35分）**

`S -> E`

`E -> Ea | aFb`

`F -> bEa | b | bEc`

1. 构造LR(1)项目集和自动机
2. 构建分析表
3. 他是LR(1)的吗？为什么？
