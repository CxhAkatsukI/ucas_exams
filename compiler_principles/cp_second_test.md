# 2026年CP第二次小测
共两大题
每年小测的第一题会变动，后面的题目比较固定

**第一题**

`S -> E`

`E -> (E)`

`E -> E + E | E - E | x`

要求设计该文法的SDD，翻译结果为从左到右依次打印x，并且附上一个嵌套深度的后缀。比如将`(x + (x) - x)`翻译为`x1 x2 x1`

**第二题**

定义了一个包含全局声明和全局Statement的类C语言，大概长这样（不保证准确）：

```
P -> D S
D -> Type idlist D | L D | epsilon
Type -> int | float | char
idlist -> id idlist | id
L -> Array id [numlist] | L id [numlist]
numlist -> num numlist | num
S -> id := E | id [numlist] := E | S S
E -> E + E | id | id [numlist]
```

其中int和float占用4字节，char占用1字节。要求每个变量对齐到4字节存储。

比如：

```
int a b
Array a [2 3] c [3 4]
float c
```

就声明了一个整形变量b，一个整形数组a和一个浮点数组c

要求给出完整的SDD方案翻译这个语言，并且同时支持：

1. 重复声明检查：监测到重复声明的时候，打印出重复的id与发生重复的行号与位置
2. 数组声明打印：按照大小打印出所有被声明的数组的信息（注意存储要求）
3. 类型检查：只有同类型的变量才可以赋值，否则报错
4. 数组引用检查：在引用数组的时候检查是否合法，否则报错。比如对于数组`a [2 3]`，引用`a [1]`和`a[4 2]`都不合法。
5. 赋值检查：变量在被使用之前应该先赋初始值，否则报错
