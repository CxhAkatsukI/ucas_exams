# 计算机组成原理 - 期末考试重点整理 (2025.7.4)

1.  **数值计算与标志位**
    *   **补码加减运算**：计算 $-68 + (-80)$ （或涉及减法）。
    *   **标志位判断**：计算结果的 **SF** (Sign Flag, 符号标志) 和 **OF** (Overflow Flag, 溢出标志)。

2.  **流水线 (Pipeline) 概念**
    *   什么是流水线？
    *   影响流水线性能的因素是什么？

3.  **指令寻址方式**
    *   **类型**：直接寻址、间接寻址、相对寻址、变址寻址。
    *   **考点**：
        1.  各种寻址方式的 **寻址范围**。
        2.  **有效地址 (EA)** 的计算公式。

4.  **流水线性能分析**
    *   画出 **流水线** 与 **非流水线** 的 **时空图**。
    *   计算 **吞吐率 (Throughput)**。

5.  **Cache (高速缓冲存储器)**
    *   **场景**：八路组相联 (8-way Set Associative) Cache。
    *   **考点**：
        1.  分析访问循环数组时的 **命中情况**。
        2.  Cache **缺失 (Miss)** 时的处理流程。

6.  **I/O 系统 - DMA**
    *   **题型**：选择题
    *   **内容**：DMA (Direct Memory Access) 控制器工作在 **\_\_\_\_\_\_** 和 **\_\_\_\_\_\_** 之间？
