# 计算机网络 2026 年期中

**除最后一题外的全部的题目均出自教材章后习题，即 计算机网络（张玉军 孟绪颖 编著）**

**第一题**

计算机网络采用层次化的细腰型体系结构，请回答以下问题：

（1）细腰对应的是哪一层？细腰结构有什么好处？

（2）这一层的核心协议是什么？起到什么作用？

**第二题**

IP地址和MAC地址是两种地址类型，请从至少三个角度比较二者的差异。

**第三题**

用户-服务模式（CS）和对等模式（P2P）是两种典型的端系统间通信模式，请分别给出两种模式的基本架构和特点。

**第四题**

信道的最高传输速率为20000码元每秒，采用多元码元的编码方式，把码元的振幅划分为16个等级，请回答以下问题：

（1）信道的数据传输速率是多少？

（2）随着码元振幅等级数量的增加，数据传输效率会提高。为什么不能通过无限制增加码元位数的方式来提高数据传输塑速率？

**第六题**

在一个采用CSMA/CD机制的网络中，链路带宽为1 Gb/s，链路长度为2km，信号传播速度为$2 \times 10^{8} m / s$。求通信协议中的最短帧长。

**第七题**

循环冗余校验是一种差错检测方法，它利用多项式除法生成校验信息。假设待发送数据对应的多项式为$R(x) = x^{5} + x^{3} + 1$，生成多项式为$G(x) = x^{3} + x^{2} + 1$。回答以下问题。

（1）请给出发送方最终需要发送的数据（以二进制比特序列的形式给出），并展示计算过程

（2）假设上一问中发送的数据在传输过程中被篡改，请给出两种篡改后的数据，使得接收方无法检测出差错

**第八题**

查错检测是计算机网络中的重要机制，请回答以下问题：

（1）查错检测机制的主要作用是什么？

（2）画图给出查错检测机制的基本原理，并进行简要解释。

（3）仅依赖查错检测可以保障数据的完整性吗？为什么？

**第九题**

停等算和滑动窗口算法是两种可靠传输机制，请回答以下问题：

（1）描述停等算法的基本原理

（2）分析停等算法在数据传输效率方面存在的问题

（3）描述滑动窗口算法的基本原理，并分析滑动窗口算法能够提升数据传输效率的原因。

**第十题**

某个网络获得了一个C类地址块202.226.192.0/24，由于网络建设需求，需要将该网络划分为14个子网，每个子网至少12台主机（子网号和主机号均不能为全0或全1）。请回答以下问题：

（1）子网的划分方案是否是唯一的？

（2）给出每个子网的子网掩码以及最多容纳的主机数

（3）分别写出第一个子网和最后一个子网的IP地址空间
