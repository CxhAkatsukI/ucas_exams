# 自然语言处理期末开卷考试题 (2026.6.23)

一、标注题（30分）

给定一段中文语料，让做：

1. 分词
2. 命名实体识别
3. 短语结构分析
4. 依存关系分析
5. 语义角色分析

二、问答题（每题5分）

1. 如何解释“语言是稳态的可遍历的随机过程”

2. 相对熵和交叉熵有什么区别

3. 简述指数图谱的构建技术与流程

4. Transformer使用的是哪种神经网络（FNN，RNN，CNN，LSTM），为什么

5. 深度学习的词向量如何表征多义现象？

三、设计题（每题15分）

1. 有一批英语篇章语料和中文篇章语料，其中的一些篇章互为彼此的翻译，设计方法把它们识别出来

2. 设计文本情感分析的流程，从数据处理到微调模型

3. 设计一个方法判断一篇文章是用LLM生成的概率有多大
